#KaherecodeChallenge

Join. Code. Share.

Apprends à programmer par la pratique. Rejoins-nous pour un challenge hebdomadaire et développe tes compétences en développement. Chaque samedi, reçois le challenge de la semaine, travaille sur le challenge et le samedi suivant on traite ensemble le challenge en live sur la chaîne Youtube de Kaherecode. Inscris-toi avec ton adresse email en remplissant le formulaire ci-dessous.

Comment ça marche?

1. Rejoins la communauté

  • Rejoins la mailing list en entrant ton adresse mail dans le formulaire ci-dessus.
  • Rejoins la discussion sur discord et discute avec la communauté.
  • Suis @kaherecode sur twitter pour ne rien rater des challenges à venir.

2. Fais le challenge

  • Chaque Samedi, tu reçois le challenge de la semaine.
  • Code le challenge, amuse toi, c'est la partie la plus fun et mets y toute ta créativité
  • N'hésite pas à demander de l'aide dans le groupe discord, on sera ravi d'aider.

3. Partage avant le live

  • Partage ton code sur Codepen, CodeSandbox ou Github avant le Samedi.
  • Partage le lien vers ton code avec le #KaherecodeChallenge ou dans le discord.
  • Le samedi on se retrouve sur la chaîne Youtube de Kaherecode pour traiter le challenge ensemble.
